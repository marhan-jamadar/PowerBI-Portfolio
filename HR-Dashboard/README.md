# HR Analytics Dashboard

## Overview
- Tracks employee headcount, attrition, attendance, and department performance
- Built using Power BI Desktop and Power Query
- Implemented DAX measures for KPIs like Attrition Rate and Average Tenure
- Data source: Sample HR dataset (non-confidential)
- Interactive visuals: Bar charts, line charts, slicers, and KPI cards
